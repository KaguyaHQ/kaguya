# All other bots (search engines, social bots) — allow public content.
#
# Thin/derivative surfaces (search, /browse filters, profile tabs, history,
# similar/ratings, edit forms, etc.) are NOT disallowed here on purpose: they
# carry a `noindex,follow` meta tag, and a crawler must be able to fetch the
# page to see it. Blocking them in robots.txt would keep already-indexed URLs
# stuck in the index as content-less stubs. Only genuinely private/no-value
# machinery is disallowed below.
